Shapes Exerice.

To run the shape exercice script, clone this repository, navigate to the fizzbuzz folder and execute:

```bash
node test.js
```

Make sure to have node installed.

## Project structure
```bash
src/
  ├── shapes/
  │   └──Cuboid.js
  │   └──Cylinder.js
  │   ├── pyramids/
  │      └──BasePyramid.js

  │      └──TriangularPyramid.js
  └── index.js
  └── BaseValidator.js
  └── ShapeFactory.js
  └── test.js
```
